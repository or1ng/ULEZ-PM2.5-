# ULEZ 政策对 PM2.5 污染的时空建模分析
项目概述

本项目利用英国 AURN 网络（2016-2024 年）的多站点空气质量监测数据，评估伦敦超低排放区 (ULEZ) 政策对 PM2.5 空气污染的影响。

该分析重点在于识别政策实施前后 PM2.5 趋势的变化，以及不同地区之间的差异。

数据

数据来源：英国城乡自动交通网络（AURN）

变量：PM2.5浓度

覆盖范围：伦敦多个监测点（2016-2024 年）

由于数据量和可重复性方面的考虑，未包含原始数据。

方法

跨多个站点的数据清理和时间对齐

ULEZ实施前后趋势分析和阶段性比较

基于回归的政策影响分析

时间趋势和区域差异的可视化

该分析是在 R 语言中实现的。

主要发现

政策实施后，核心超低排放区（ULEZ）的PM2.5水平呈现明显的下降趋势。

不同地区的改善幅度各不相同。

研究结果表明，政策效果在空间上呈现异质性而非均匀性。

项目目的

该存储库是作为硕士毕业项目的一部分而创建的，并共享以展示数据分析和建模技能。
