# 伦敦 ULEZ 政策对 PM2.5 污染影响分析

## 项目概述
本项目基于英国官方空气质量监测网络（AURN）的数据，对伦敦 Ultra Low Emission Zone（ULEZ）政策实施前后 PM2.5 污染水平的变化情况进行分析。

项目重点关注两个问题：  
1）ULEZ 政策实施后，PM2.5 浓度是否出现明显变化；  
2）不同监测站点之间，政策影响是否存在差异。

该项目为本人硕士毕业设计的一部分，主要用于展示数据分析与建模能力。

---

## 数据说明
- **数据来源**：英国 Automatic Urban and Rural Network（AURN）  
- **核心指标**：PM2.5 浓度  
- **时间范围**：2016–2024 年  
- **空间范围**：伦敦多个空气质量监测站点  

出于数据体量和复现性的考虑，本仓库未直接包含原始数据。相关数据可通过 AURN 官方渠道获取。

---

## 分析流程
项目整体按照完整的数据分析流程开展，主要包括以下步骤：

1. **数据预处理**
   - 原始监测数据的清洗与格式统一  
   - 缺失值处理与多站点时间序列对齐  

2. **探索性分析**
   - PM2.5 浓度随时间变化的趋势分析  
   - 政策实施前后污染水平的对比观察  

3. **建模与评估**
   - 基于回归方法的政策效果分析  
   - 非线性趋势建模，用于刻画长期变化特征  
   - 不同监测站点之间结果的对比分析  

4. **结果可视化**
   - PM2.5 时间序列趋势图  
   - 政策前后及区域差异对比图  

项目分析过程全部使用 **R 语言** 完成。

---

## 主要结论
- ULEZ 核心管控区域在政策实施后，PM2.5 浓度整体呈下降趋势  
- 不同监测站点的改善幅度存在差异，政策效果具有一定空间异质性  
- 结果表明，政策影响并非在所有区域均匀发生，本地因素对空气质量变化仍具有重要作用
