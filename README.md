# ULEZ-PM2.5-
围绕伦敦 ULEZ 政策的实施效果，基于英国 AURN 多站点空气质量数据（2016–2024），构建时间序列与空间分析相结合的建模框架，研究 PM2.5 污染变化的趋势及区域差异。

在数据层面，对多站点长时间序列进行了清洗、插补与结构化处理；在建模层面，结合趋势分解、Difference-in-Differences（DID）和广义加性模型（GAM），刻画政策实施前后的非线性变化特征，并通过空间插值分析污染分布差异。

结果表明，政策实施后核心区域 PM2.5 浓度下降较为明显，但模型结果对方法设定和区域划分较为敏感，不同区域的改善程度存在差异，反映出政策效果具有一定的空间不均衡性，体现了政策评估中时空异质性和模型选择的重要性。
